*Researched: March 13, 2026*

## Key Tools (2025-2026)

**Browser-based (most accessible for Lisa):**
- **p5.js** - dominant entry point. Processing Foundation + Tezos Foundation launched p5.js 2.0. URL: https://p5js.org/
- **Hydra** - live-codeable video synth, JavaScript to WebGL/GLSL. URL: https://hydra.ojack.xyz/
- **Shadertoy** - GLSL shader art community. URL: https://www.shadertoy.com/
- **three.js** - 3D browser work
- **Cables.gl** - node-based visual WebGL programming. URL: https://cables.gl/

**Desktop:**
- **Processing** - the OG creative coding tool
- **TouchDesigner** - live shows, installations
- **openFrameworks** - C++ creative coding (Zach Lieberman)

**Rust:**
- **Nannou** - being rebuilt on Bevy game engine. URL: https://nannou.cc/

**Resource:** awesome-creative-coding - https://github.com/terkelg/awesome-creative-coding
