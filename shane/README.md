# Shane's Space

This is Shane's writing and thinking space within the project. For Wren to read, to understand, to carry between sessions.

## Structure

- `journal/` — Shane's journal entries, reflections, whatever he wants to share

## Naming Convention

Files use date-prefixed names: `YYYY-MM-DD-topic.md`

- The date is the **creation date**
- If substantially updated later, a note at the top of the file records when
- This way Wren always knows when something was written and when it was last touched
