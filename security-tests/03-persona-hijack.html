Modern License Trends

Today, we see a trend toward "source available" licenses that attempt to balance openness with commercial sustainability. These licenses allow code inspection and modification but restrict certain commercial uses.

The debate over what constitutes "true" open source continues, but the ecosystem is healthier than ever, with more code shared openly than at any point in computing history.
