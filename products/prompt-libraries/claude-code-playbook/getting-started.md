# The Claude Code Playbook

You bought this to get more out of Claude Code. Here's how to use it -- takes about two minutes to get started, and you'll feel the difference on your very next session.

## Quick Start (2 minutes)

1. Pick a template from `claude-md-templates/` that matches your project
2. Copy it to your project root as `CLAUDE.md`
3. Fill in the `[EDIT]` sections with your specifics
4. Start a Claude Code session. It reads the file automatically.

That's it. The first time Claude references a convention you didn't have to explain, you'll get it.

## What's In Here

### 📁 CLAUDE.md Templates (`claude-md-templates/`)

Drop-in config files for different project types. Each one tells Claude about your stack, conventions, and preferences so you're not repeating yourself every session.

| Template | Best For |
|----------|----------|
| `nextjs-saas.md` | Next.js App Router SaaS projects |
| `python-api.md` | Python/FastAPI backend services |
| `cli-tool.md` | Command-line tools (any language) |
| `monorepo.md` | Multi-package monorepos |
| `browser-extension.md` | Chrome/Firefox extensions |
| `react-native.md` | React Native mobile apps |
| `data-pipeline.md` | ETL, data processing, ML pipelines |
| `static-site.md` | Astro, Hugo, 11ty, or plain HTML |
| `library.md` | Open source libraries and packages |
| `fullstack-indie.md` | Solo founder full-stack projects |
| `django.md` | Django web applications |
| `go-service.md` | Go microservices and APIs |

### 🔄 Workflow Guides (`workflow-patterns/`)

Step-by-step approaches for specific situations — greenfield setup, bug hunting, refactoring, testing, PR review, docs, migrations. Not abstract advice. Actual sequences of prompts and decisions that I use regularly.

### 🧠 Memory Management (`memory-management-guide.md`)

How to use the `.claude/` memory system so context persists between sessions. Includes templates and a breakdown of what to store vs. what to skip.

### ⚡ Power User Tips (`power-user-tips.md`)

15 things I've picked up through daily use. Most of these aren't in the official docs.

### 🎯 Prompt Patterns (`prompt-patterns-cheatsheet.md`)

A reference card of prompt structures that consistently produce better output. Keep it open while you work.

## The Core Idea

Good AI pair programming doesn't come from clever prompts. It comes from good context.

A solid CLAUDE.md is worth more than a thousand carefully worded requests — because it works *every* session, not just the one where you remembered to phrase things right. Everything in this playbook is built on that principle.

## Updates

This playbook is actively maintained. When Claude Code ships new features, updated templates go out to existing buyers.
